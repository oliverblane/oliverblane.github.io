Who am I?

Hi, I'm Oliver, a second-year Physics student at Queen Mary Univesity of London - and I love to code! Feel free to skip the pleasantries and take a look at my projects. All of my projects are available on my GitHub, where you will find a much more detailed description of each project.

This is a portfolio containing my favourite projects that I have worked on, in fields ranging from data science, to scientific computing, to mobile development (and web development if you include this portfolio).

During these projects, I have used languages such as Python, JavaScript, HTML and CSS. I have also learnt to use the React Native framework for mobile development, and applications such as Jupyter Notebook (for Python) and R Studio (for R) for data analysis.

Estimating a daily Rt value for COVID-19

When COVID-19 began to sweep across the UK, there was huge emphasis on the R number. Government officials insisted that we must get R below 1, and keep it there. No exact values were ever broadcasted, only wide ranges in which R may lie, and so I decided try and estimate R (within error) for each day. The figure on the left shows some of my key findings for the UK up until the middle of May.

Transition temperature of dielectrics using quasielastic neutron scattering

Dielectric materials are used in all sorts of electrical components. I investigated at which temperature the behaviour of these dielectrics change, and what effect this has on the energy barrier to molecules within the dielectrics.

Analysing Lego data from the last 70 years

Ever wondered how the number of pieces of Lego per set has changed over the last 70 years? I did, and so I used some Lego data to find the answer (along with the answer to some other questions... See my GitHub for more!).

Track - the workout tracker that focuses on data analysis

Fitness is a passion of mine, and so I decided to start developing a mobile app that meets my workout needs. This fitness tracker is being designed for those who want the most detailed analysis of their workouts. This is an ongoing project that I work on during my spare time.
